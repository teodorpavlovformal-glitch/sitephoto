## Deploy to Vercel (free, ~1 min)

### Option A — Upload (easiest)
1. Go to [vercel.com](https://vercel.com) → sign up free with Google
2. Click **Add New Project** → **Upload**
3. Drag and drop the entire `pavlov-photography` folder
4. Click **Deploy**
5. Done — you get a `*.vercel.app` URL instantly ✅

### Option B — GitHub (recommended for updates)
1. Create a free [GitHub](https://github.com) account
2. Create a new repo, upload all 4 files
3. In Vercel → **Add New Project** → **Import Git Repository**
4. Select your repo → Deploy
5. Every future edit to GitHub auto-deploys to Vercel ✅

## Custom domain (optional)
After deploying, go to: Project → Settings → Domains → add `pavlovphoto.bg` or similar.
